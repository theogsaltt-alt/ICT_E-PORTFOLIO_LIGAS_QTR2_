<div class="textblock">
<h1>CSS Rule Structure</h1>
<p>A CSS rule consists of two main parts:</p>
<ul>
<li>Selector: defines which HTML element(s) the rule applies to.</li>
<li>Style Declarations: define how those elements should appear.</li>
</ul>
</div>

<div class="textblock">
<h1>Basic Format of a CSS Style Rule</h1>
<ul>
<li>For Inline Style Rule:</li>
</ul>
<pre><code>&lt;tag style="style declaration"&gt;
&lt;tag style="property1:value; property2:value; property3:value;"&gt;
</code></pre>
<ul>
<li>For Document-Level and External Style Sheets:</li>
</ul>
<pre><code>selector { style declarations }
selector { property1:value; property2:value; property3:value; }
</code></pre>
</div>
</div>
</div>
</div>


<div class="containerlessons">
<div class="contentvertical">
<div class="textblock">
<h1>Style Properties</h1>
<p>CSS style properties are grouped into six main categories:</p>
<ul>
<li>Fonts</li>
<li>Colors and Backgrounds</li>
<li>Text</li>
<li>Boxes and Layout</li>
<li>Lists</li>
<li>Tag Classifications</li>
</ul>
</div>

<div class="textblock">
<h1>Classes: What are they?</h1>
<p>A classis a user-defined selector used to apply a specific style to one or more HTML elements.</p>
<ul>
<li>Used to apply a style rule to specific HTML tags.</li>
<li>Controls individual HTML element formatting.</li>
<li>Reusable style definitions — one class can style multiple elements.</li>
</ul>
</div>
</div>
</div>

<div class="containerlessons">
<div class="contentvertical">
<div class="textblock">
<h1>Defining Classes in CSS</h1>
<ul>
<li>Classes are defined using a period (<code>.</code>).</li>
<li>Two methods to write a class:
<ol>
<li>Element + Class: <code>p.golden { color: blue; }</code></li>
<li>Class Only: <code>.sodapop { color: red; }</code></li>
</ol>
</li>
<li>In HTML, apply a class using the <code>class="classname"</code> attribute.</li>
</ul>
</div>

<div class="textblock">
<h1>Examples</h1>
<img src="classexamples.png" alt="class examples" style="border-radius:15px; width: 500px;">
<p>Note: One element can have multiple classes, and one class can be applied to many elements.</p>
</div>
</div>
</div>

<div class="containerlessons">
<div class="contentvertical">
<div class="textblock">
<h1>Class Selector</h1>
<p>A <strong>Class Selector</strong> applies styles to elements with a specific class name.</p>
<h3>Element + Class Selector</h3>
<p>This selector targets only a specific element with a given class name.</p>

      <pre><code>p.note {
  color: blue;
  font-size: 18px;
}

</code></pre>
</div>
<div class="textblock" class="table">
<table border="1" cellpadding="8" cellspacing="0">
<tr>
<th>Selector Type</th>
<th>Example</th>
<th>Description</th>
</tr>
<tr>
<td>Class Selector</td>
<td><code>.classname</code></td>
<td>Applies to <strong>any element</strong> with that class.</td>
</tr>
<tr>
<td>Element + Class Selector</td>
<td><code>p.classname</code></td>
<td>Applies only to that <strong>specific element</strong> with that class.</td>
</tr>
</table>

<p><strong>Summary:</strong></p>
<ul>
 <li><code>.classname</code> → any element with that class.</li>
<li><code>element.classname</code> → only that element type with that class.</li>
</ul>
</div>
</div>
</div>

<div class="containerlessons">
<div class="contentvertical">
<div class="textblock">
<h1>Grouping Selectors</h1>
<p>You can group multiple selectors in a single CSS rule to apply the same styles to different elements.</p>
<h1>Sample Code</h3>
<pre><code>/* Instead of writing: */
h1 { color: blue; }
p { color: blue; }

/* You can group them like this: */
h1, p { color: blue; }
</code></pre>

<p><strong>Tip:</strong> Grouping selectors reduces redundancy and keeps your CSS clean and efficient.</p>
</section>
</div>
</div>
</div>

<div class="textblock">
<h1>When to Use Each Selector</h1>
<table border="1" cellpadding="8" cellspacing="0" class="table">
<tr>
<th>Selector Type</th>
<th>When to Use</th>
</tr>
<tr>
<td><strong>Class Selector</strong></td>
<td>When you want to apply the same style to multiple elements that share a common purpose.</td>
</tr>
<tr>
<td><strong>ID Selector</strong></td>
<td>When styling a unique element that appears only once on a page (e.g., header, footer).</td>
</tr>
<tr>
<td><strong>Element Selector</strong></td>
<td>When styling all occurrences of a specific HTML tag (e.g., all &lt;p&gt; or &lt;h1&gt;).</td>
</tr>
<tr>
<td><strong>Universal Selector</strong></td>
<td>When you want to style all elements at once, usually to reset defaults (margins, padding).</td>
</tr>
<tr>
<td><strong>Group Selector</strong></td>
<td>When multiple elements share the same style — helps reduce redundancy.</td>
</tr>
<tr>
<td><strong>Attribute Selector</strong></td>
<td>When styling elements based on their HTML attributes or attribute values.</td>
</tr>
</table>

<h1>Setting Dimensions</h1>
<p>CSS can be used to manage dimensions such as visibility, width, or height of HTML elements.</p>
<h1>Properties for Dimensions</h1>
<table border="1" cellpadding="8" cellspacing="0" class="table">
<tr>
<th>Property</th>
<th>Definition</th>
<th>Example</th>
</tr>
<tr>
<td>visibility</td>
<td>Indicates if an element is visible or hidden</td>
<td>
visibility: visible: default value, element is visible<br>
visibility: hidden: element is hidden (still takes up space)<br>
visibility: collapse: used on table rows, columns, etc.
</td>
</tr>
<tr>
<td>width</td>
<td>Indicates the width</td>
<td>width: 200px</td>
</tr>
<tr>
<td>height</td>
<td>Indicates the height</td>
<td>height: auto</td>
</tr>
<tr>
<td>line-height</td>
<td>Indicates line spacing</td>
<td>line-height: 12px</td>
</tr>
<tr>
<td>max-height</td>
<td>Indicates the maximum height</td>
<td>max-height: 12px</td>
</tr>
<tr>
<td>min-height</td>
<td>Indicates the minimum height</td>
<td>min-height: 12px</td>
</tr>
<tr>
<td>max-width</td>
<td>Indicates the maximum width</td>
<td>max-width: 12px</td>
</tr>
<tr>
<td>min-width</td>
<td>Indicates the minimum width</td>
<td>min-width: 12px</td>
</tr>
</table>
<p>This CSS feature is useful for controlling paragraphs, tables, and other elements.</p>
</div>
</div>

<h1>Pseudo-Classes and Links</h1>
<p>Pseudo-classes are predefined selectors (not user-defined like classes) that apply styles to elements in specific states.  For example, the anchor element (&lt;a&gt;) supports pseudo-classes like :link, :visited, :hover, and :active.</p>
<h1>Anchor Pseudo-Classes</h1>
<ul>
<li>link: for unvisited hyperlinks.</li>
<li>visited: for visited hyperlinks.</li>
<li>hover: for when the mouse hovers over the link.</li>
<li>active: for when the link is being clicked.</li>
<li>focus: for focused link elements.</li>
</ul>
</div>
</div>
</div>

<div class="containerlessons">
<div class="contentvertical">
<div class="textblock">
<h1>Syntax Example</h1>
<table class="table">
<tr><th>Property</th><th>Example</th><th>Description</th></tr>
<tr><td>:link</td><td>a:link { … }</td><td>Sets unvisited link characteristics.</td></tr>
<tr><td>:visited</td><td>a:visited { … }</td><td>Sets styles for visited links.</td></tr>
<tr><td>:hover</td><td>a:hover { … }</td><td>Sets styles when hovering over a link.</td></tr>
<tr><td>:active</td><td>a:active { … }</td><td>Sets styles when the link is clicked.</td></tr>
<tr><td>:focus</td><td>a:focus { … }</td><td>Sets styles for focused links.</td></tr>
</table>
<p><strong>Remember:</strong> The correct order is LVHA — Link, Visited, Hover, Active.</p>
</div>
</div>
